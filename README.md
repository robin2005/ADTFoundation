## Require

- iOS 8.0+ / macOS 10.10+ / tvOS 9.0+ / watchOS 2.0+
- Swift 4.0

## Install

CocoaPod

```

pod 'ADTFoundation'

or pod 'ADTFoundation/Data'

or pod 'ADTFoundation/Date'

or pod 'ADTFoundation/Number'

or pod 'ADTFoundation/Data'

or pod 'ADTFoundation/Tools'

or pod 'ADTFoundation/URL'

or pod 'ADTFoundation/EventBus'

```
